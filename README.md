# LMS AI Project

An AI-powered Learning Management System (LMS) chatbot designed to assist students and users with learning-related queries through an interactive web interface.

## Features

* Interactive chatbot interface
* User-friendly frontend design
* Backend integration using Python
* Responsive web interface
* Docker configuration support

## Project Structure

```text
backend/
├── main.py
├── requirements.txt
└── utils.py

frontend/
├── index.html
└── static/
    ├── script.js
    └── style.css

compose.yaml
README.md
```
